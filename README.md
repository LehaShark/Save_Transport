# Save_Transport!
Частота дискретизации звука была взята 44.1К
В датасете в данный момент использовано разбиение сэмплов по секунде и переведено в спектрограммы в формате jpg.
Из класса "cut" были взяты для валидационной выборки сэмплы: 17, 21, 29, 36, 41, 58. Тем самым после разделения посекундно получается 30 картинок. Остальные пошли на тренировочную выборку.
Из класса noise были взяты первые 234 сэмпла по 5 секунд, разделены посекундно и отправлены в тренировучную выборку. Последующие 6 сэмплов разделены посекундно и отправлены на валидационную выборку.
